# Get

## Abi(Get contract ABI)

#### url
- /get/abi/<string:contract_name>

#### method
- GET

#### doc
```
    Gets the ABI of the contract based on the param.

    :param
        contract name

    :return:
        { 'result': 0/1, 'msg': the contract abi file }
```

- POST

#### doc
